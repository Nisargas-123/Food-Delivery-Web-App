# Food Delivery Web Application

## Project Overview
This is a Full Stack Food Delivery Web Application developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The application allows users to browse food items, add them to the cart, place orders, and manage their accounts.

## Features
- User Registration and Login
- Browse Food Menu
- Add Items to Cart
- Place Orders
- Order Management
- Responsive User Interface

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- JavaScript
- HTML
- CSS

## Project Structure
- frontend/ - React application
- backend/ - Express server and APIs
